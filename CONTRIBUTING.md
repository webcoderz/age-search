# Contributing

Thanks for helping improve `age_search`.

## Development setup

- Python: 3.10+
- Install dev deps:

```bash
pip install -e ".[dev]"
```

## Running checks

- Tests:

```bash
pytest
```

- Lint:

```bash
ruff check .
```

## Pull requests

- Keep PRs focused and small when possible.
- Add/adjust tests for behavior changes.
- If you change public APIs, update the README and `CHANGELOG.md`.
